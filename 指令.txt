請撰寫一個c語言程式，依照以下格式發送API請求給open ai api，並且將api回覆內容(用cjson提取回應)製作成一個檔案，名為ai_script.toml。預設已安裝cjson、curl。

api請求/回覆格式(toml檔案內只需要有assistant中的text(例子中的"Hello! How can I assist you today?"))：
curl https://api.openai.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $OPENAI_API_KEY" \
  -d '{
  "model": "gpt-3.5-turbo-16k",
  "messages": [
    {
      "role": "system",
      "content": [
        {
          "type": "text",
          "text": "test"
        }
      ]
    },
    {
      "role": "user",
      "content": [
        {
          "type": "text",
          "text": "this is the input messege"
        }
      ]
    },
    {
      "role": "assistant",
      "content": [
        {
          "type": "text",
          "text": "Hello! How can I assist you today?"
        }
      ]
    }
  ],
  "temperature": 1,
  "max_tokens": 4095,
  "top_p": 1,
  "frequency_penalty": 0,
  "presence_penalty": 0
}'
